# General-Purpose-Pick-and-Drop-Mobile-Robot


## Overview

This project presents the design and development of a general-purpose pick-and-drop mobile robot capable of moving, navigating, and manipulating objects using a robotic gripper.

The robot was designed as part of an Engineering Workshop project using Arduino, servo motors, DC motors, and a custom mechanical structure.

---

## Features

- Mobile robot platform
- Robotic gripper
- Forward and backward movement
- Left and right turning
- Servo-controlled gripper
- Arduino-based control system
- Modular design

## Working Principle

The robot receives movement commands through digital input pins.

The Arduino controls two DC motors for movement and a servo motor for operating the robotic gripper.

The system can:

- Move Forward
- Move Backward
- Turn Left
- Turn Right
- Open Gripper
- Close Gripper
